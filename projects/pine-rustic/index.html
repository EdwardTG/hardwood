Rustic Pine Look

A charming rustic pine flooring installation that brings warmth and character to this country home. Using reclaimed pine wood, we created a floor with authentic vintage appeal while ensuring modern durability.

Project Specifications

  • Material: Reclaimed pine wood
  • Pattern: Random width planks
  • Finish: Hand-scraped with vintage patina
  • Area: 800 sq ft
  • Duration: 12 days

Project Process

  1. Selection of reclaimed pine boards
  2. Careful cleaning and preparation of materials
  3. Subfloor assessment and reinforcement
  4. Layout planning for random width pattern
  5. Hand-scraping and distressing
  6. Installation with vintage techniques
  7. Custom aging and patina application
  8. Protective finish application
  9. Final aging touches and inspection
